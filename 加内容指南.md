# 课题组网站维护指南

本指南教你如何用 Claude Code 往课题组网站添加内容。整个过程在对话中完成，无需手动编辑文件。

## 准备工作

1. 安装 Claude Code（见 https://docs.anthropic.com/en/docs/claude-code）
2. 克隆仓库：

```
git clone https://github.com/SDU-WH-GraphTheory/SDU-WH-GraphTheory.github.io.git
cd SDU-WH-GraphTheory.github.io
```

3. 在该目录下启动 Claude Code，然后按下面的场景操作。

---

## 场景一：发一篇新闻 / 博文

直接对 Claude Code 说：

> 帮我创建一个新博文，标题是「XXX学术会议参会总结」，作者是我（用户名XXX），内容是：……

Claude 会在 `content/post/` 下创建你的文章。**格式要求**：文章支持 Markdown + LaTeX 数学公式（`$...$` 或 `$$...$$`）。

创建完后，说：

> 提交并推送到 GitHub

---

## 场景二：添加一篇论文

> 帮我添加一篇论文：标题「XXX」，作者宋慧敏、XXX，发表在《Discrete Mathematics》2025年，期刊论文。

Claude 会在 `content/publication/` 下创建该论文页面。

---

## 场景三：创建一个活动/讨论班（有海报）

> 我在 Downloads/Post/ 文件夹里放了讨论班海报的 PDF，帮我给每张海报创建一个大事记文章。海报图片要嵌入正文，也作为封面 banner。

Claude 会：
1. 从海报 PDF 中提取文字（OCR）
2. 把 PDF 转成 PNG 放到 `static/uploads/posters/`
3. 在 `content/event/` 下给每张海报创建文章

**如果你没有海报**，直接说：

> 帮我创建一个活动：「图论讨论班第X期」，时间是 2026 年 5 月 10 日下午 2 点到 4 点，地点数学与统计学院 XXX 会议室，报告人是 XXX。

### 大事记文章格式规范

新增事件时请遵循以下格式（见 `content/event/` 下已有文章）：

- **文件夹命名**：`YYYY-MM-DD-session-XX-slug/`
- **每个文件夹包含**：`index.md` + `poster.png`（海报作为封面图）
- **海报 PNG**：同时存放在 `static/uploads/posters/` 中以供正文嵌入
- **文章排序**：按讨论班日期排序（由 frontmatter 中的 `date` 字段决定）
- **LaTeX 数学公式**：使用 `\(...\)` 行内 / `\[...\]` 行间格式

**Frontmatter 模板**：

```yaml
---
title: "讨论班主题标题"
event: "图论与组合数学讨论班"
event_url: ""
location: 北衡楼 1216
address:
  city: 威海
  region: 山东
  country: 中国
summary: 简短摘要
abstract: 详细摘要
date: 'YYYY-MM-DDTHH:MM:00+08:00'
date_end: 'YYYY-MM-DDTHH:MM:00+08:00'
all_day: false
publishDate: 'YYYY-MM-DDTHH:MM:00+08:00'
authors: [主讲人姓名]
featured: false
image:
  caption: '第X次讨论班海报'
  focal_point: Right
  filename: poster.webp
math: true
---
```

**正文结构**：基本信息（列表）→ 内容概要（段落）→ 海报（图片嵌入）

## 场景四：更新自己的个人页面

> 帮我更新我的个人页面，用户名是「路宇轩」。在 bio 里加上我最近的研究方向，interests 加一个「极值图论」。

Claude 会修改 `content/authors/路宇轩/_index.md`。

---

## 场景五：修改导师信息

> 帮我把宋慧敏老师主页上的 XXX 改成 YYY。

---

## 最后一步：提交并推送

任何修改完成后，说：

> 提交并推送到 GitHub

网站会自动部署，几分钟后生效。

---

## 文件对应表

| 网站上看到的 | 仓库位置 |
|---|---|
| 关于我们 | `content/tour/index.md` |
| 近闻（新闻列表） | `content/post/` |
| 成员（团队页面） | `content/people/index.md` |
| 成员个人页 | `content/authors/<姓名>/_index.md` |
| 大事记（活动） | `content/event/` |
| 文章（论文列表） | `content/publication/` |
| 联系方式 | `content/contact/index.md` |
| 网站设置 | `config/_default/` |

---

## 注意事项

- 所有内容用 Markdown 写，支持 LaTeX 数学公式
- 提交前让 Claude 帮你预览一下修改内容
- 推送后等 2-3 分钟，GitHub Actions 会自动部署
- 有问题直接在对话里问 Claude Code
